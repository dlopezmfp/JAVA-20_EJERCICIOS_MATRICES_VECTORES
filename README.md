# EJERCICIOS DE REPASO BUCLES / MÉTODOS

1. Crear un programa que gestione una lista de nombres de productos y sus precios
   en arrays paralelos, utilizando métodos void (procedimientos) para las tareas
   “aplicar descuento” (el usuario elegirá el descuento a aplicar y se pasará como
   parámetro junto al vector) y “mostrar inventario” (se mostrará cada producto
   junto a su precio).

2. Crear un programa que analice un array de temperaturas, utilizando funciones
   (return) para calcular valores estadísticos (define un array con las 10 últimas
   temperaturas registradas en tu ciudad y crea funciones para calcular el promedio
   de las temperaturas, el máximo y cuántas temperaturas hay superiores al
   promedio, y mostrarlas. Las funciones

3. Crea un programa que defina una matriz con las notas de 4 alumnos distintos en
   3 asignaturas distintas. Con un array paralelo de tamaño 4 define el nombre de
   los 4 alumnos y uno de tamaño 3 con el nombre de las asignaturas. A
   continuación, tendrás que definir una función para mostrar cada alumno junto a
   su nota en cada asignatura. Realizarás otra función para calcular la nota media
   de cada alumno, otra para averiguar cuál es el mejor alumno en cada asignatura,
   otra para calcular cuál es la asignatura que mejor se le da a cada alumno.

4. Crea una matriz de 3 x 3 y utiliza funciones para devolver la suma de cada fila, de
   cada columna y de sus dos diagonales.

5. Crea una matriz de 5 x 5 con caracteres en donde una de las casillas tenga la letra
   ‘T’ correspondiente a un tesoro. El usuario tendrá 4 intentos para averiguar
   dónde está el tesoro, por cada intento que falle se le indicará dos frases dando
   pistas (comentando si el tesoro está por encima, en la misma línea o debajo
   donde está buscando, o si está a la izquierda, derecha o misma columna). Deberá
   haber como mínimo una función o procedimiento para crear la matriz situando
   el tesoro de forma aleatoria y otra función para decir al usuario si ha acertado o
   fallado, y dar las pistas adecuadas.

6. Crea una matriz de 4 x 4 y mediante funciones o procedimientos debes buscar si
   un número está dentro de la matriz y mostrar de forma ordenada cada fila o
   columna.

7. Dado un array de enteros, crea dos funciones que devuelvan el valor máximo y
   el mínimo del array.

8. Escribe una función que reciba un array y cuente cuántos números pares
   contiene.
9. Crea un método que rote un array N posiciones hacia la derecha. Ejemplo:
   [1,2,3,4,5] → rotar 2 → [4,5,1,2,3].
10. Función que devuelva cuántas veces aparece un número en un array.
11. Función que cuente cuántos ceros hay en una matriz.
12. Función que reciba una matriz y un número de fila y devuelva la suma total de
    esa fila.
13. Crea una función que reciba un array de enteros y devuelva un nuevo array con
    los elementos en orden inverso (no utilizar el método reverse).
14. Crea una función que reciba un array de enteros y devuelva un booleano (true o
    false) indicando si el array está ordenado de forma ascendente.
15. Crea un procedimiento que reciba una matriz de N x M y la rellene con números
    consecutivos, empezando desde 1.
16. Crea una función que reciba una matriz cuadrada y devuelva el promedio de los
    elementos de su diagonal principal.
17. Crea una función que reciba un array de String y devuelva el número total de
    vocales (a, e, i, o, u) que hay en todas las cadenas combinadas.
18. Define un array con las pulsaciones que has tenido en los últimos 15
    entrenamientos. Crea funciones para calcular:
    a. Frecuencia media
    b. Cuántas sesiones estuvieron en zona aeróbica (por encima 120)
    c. El día con mayor pulsación
19. Una matriz NxM donde N es número de alumnos y M número de propuestas.
    Cada posición indica votos (0 o 1). Crea un array paralelo de String de tamaño N
    con los nombres de todos los alumnos/as y otro de tamaño M con el nombre de
    los N alumnos/as que se han presentado a delegado/a. Crea funciones para:
    a. Mostrar la matriz
    b. Contar cuántos votos tiene cada propuesta (indicando los nombres)
    c. Averiguar cuál fue la más votada
    d. Mostrar los votos de cada alumno (indicando el nombre de quién vota y
    el nombre de los que ha votado).

20. Define un array de nombres de alumnos y otro de booleanos indicando si han
    asistido. Crea funciones para:
    a. Mostrar la lista
    b. Contar asistencias
    c. Mostrar los que faltaron
